ECE326: Programming Languages, Fall 2020

Course Description

General introduction of modern programming languages and paradigms, including imperative programming, object-oriented programming, meta programming, concurrent programming and functional programming. The course will be supplemented by coding assignments using popular programming languages including C++11, Python, and Rust.

Note that the course description on the academic calendar is not up-to-date.

Course Objectives
  • Identify and analyze different programming paradigms
  • Learn C++11, Python, and Rust
  • Have deep understanding of advanced programming concepts
  • Be able to solve problems with different programming techniques and evaluate their strengths and weaknesses

Policy on Cheating

Plagiarism and cheating WILL NOT be tolerated under any circumstances. Cheating includes, but is not limited to, generating expected output in an attempt to fool the automated marking script, or using unauthorized aids in assignments and exams. It is also an offense to allow a copy of your work to be submitted by another person for credit, whether with or without your knowledge. As such, it is your responsibility to protect your work from being copied by others.

All assignments submitted for credit in this course will be compared pairwise to attempt to identify cases of collusion, copying, and similar offenses. The comparison is performed by a sophisticated program that is capable of detecting similar programs even if considerable effort has been taken to conceal their similarity.

Any work submitted for credit that is not the work of the person(s) submitting it will be treated as an offense under the Code of Academic Discipline of the University. The Code of Academic Discipline will be rigidly enforced in this course. Penalties can range from grade penalties in the course to suspension from the university. The Dean, as advised by the instructors and the Departmental Chair, determines the penalty for each case.

For more details, please refer to the Academic Integrity web site at the University.
